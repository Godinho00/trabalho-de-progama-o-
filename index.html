LAÇOS DE REPETIÇÃO: Laços de repetição, ou loops, são estruturas de programação que permitem executar um bloco de código várias vezes. São fundamentais para criar fluxos dinâmicos, onde o comportamento muda de acordo com as entradas e condições encontradas.

LINGUAGENS DE PROGRAMAÇÃO: Linguagem de programação é um conjunto de regras e símbolos que permite criar instruções para um computador. Essas instruções são usadas para desenvolver programas, aplicativos e outras soluções tecnológicas.

PRINCIPAIS: Python, JavaScript, Java, C#, C++, PHP, TypeScript, Shell

CARACTERISTICAS: Cada linguagem tem suas particularidades e é mais usada para um tipo de solução

EXEMPLOS: Ruby, Swift, R, Assembly, C, C++

USOS: Desenvolvimento de aplicativos, jogos, sistemas operacionais, controladores, utilitários

PRINCIPAIS TAGS UTILIZADAS: As principais tags HTML são usadas para definir parágrafos, títulos, listas, formulários, imagens e links.

TAGS DE TEXTO: -p: Define um parágrafo -br: Insere uma quebra de linha -hr: Define um filete horizontal -h1: até -h6: São tags de título que destacam conteúdos ou subdividem seções de texto -strong: Destaca um conteúdo com muita importância -em: Marca um texto que se quer enfatizar

TAGS DE IMAGENS: -img: Insere uma imagem em um documento da web -area: Define uma área de um mapa de imagem -map: Define um mapa de imagem

TAGS DE LISTA: -ul: Define uma lista desordenada -ol: Define uma lista ordenada -li: Define cada item de uma lista, seja ela ordenada ou não

Tags de formulário -form Define um formulário no HTML, -input Define um campo de entrada de dados, -button Define um botão, -select Define uma caixa de seleção.

OUTRAS TAGS: -base: Define a URL base para todas as URLs relativas no documento -link: Vincula folhas de estilo externas -meta: Fornece metadados sobre o documento HTML -span: Agrupa elementos em linha para aplicar estilos -table: Cria tabelas

As tags podem ser categorizadas em "nível de bloco" (block-level) e "em linha" (inline).
